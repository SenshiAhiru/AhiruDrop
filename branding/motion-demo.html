01 · Coin Idle Float

Mascote vivo em todas as telas

Loop suave de 4 segundos com balanço lateral, sombra dinâmica e glow dourado. Pra cabeçalho de saldo, dashboard, página de depósito.

02 · Number Counter

Saldo conta como em slot machine

Quando depósito completa via webhook, o saldo no header anima do valor antigo até o novo. Visual gamificado, reforça a sensação de "ganhei". Easing personalizado pra dar peso.

Cards entram em cascata ao scroll

05 · Skeleton Shimmer

Carregamento elegante

Enquanto a API responde, skeletons com shimmer dourado-roxo dão a impressão de movimento e mantêm o layout estável (zero CLS).

Sorteio como animação narrativa

Página de verificação anima cada passo da prova matemática: server seed revela, block hash chega, HMAC computa, número sai. Vira "show", não documentação chata.
