# v1 Reference — AI-generated vision board

Pasta pra guardar a **vision board** do brand v1 (o composite AI-gerado aprovado em 2026-04-29) e quaisquer assets isolados que você já tenha.

## Como guardar aqui

Quando tiver:

- **Composite v1 completo** (a imagem grande com tudo) → salva como `vision-board.jpg` ou `vision-board.png`
- **Ícone da moeda AHC isolado** (você disse que tem) → salva como `coin-icon.png` (com transparência, alta resolução)
- **Quaisquer outros assets isolados** que vierem por aí → nomeia descritivamente (`character-anime.png`, `wordmark-light.png`, etc.)

## Por que importa

Esses arquivos são a **referência visual aprovada**. Quando você contratar designer profissional pra produzir os PSDs/AI source files, ele vai precisar:

1. Olhar essa vision board pra entender o estilo
2. Usar o `coin-icon.png` como base (já está na resolução que você precisa)
3. Recriar os outros elementos (logo wordmark, banners, character poses) **mantendo a estética**

## ⚠️ Limitações da vision board

- **AI-gerada, sem source files** — não dá pra editar diretamente. É referência pra recriação.
- **Personagem anime tem só 1 pose** — designer precisa criar variações (vencedor, surprised, sleeping, pointing) mantendo o mesmo personagem reconhecível.
- **Texturas/efeitos** (lightning, partículas) precisam ser refeitos no Photoshop/Illustrator pelo designer.
